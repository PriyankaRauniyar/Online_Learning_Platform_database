# Online Learning Platform – MySQL Database Project

This repository contains a MySQL-based relational database system developed as part of a database foundations project.
The focus is on database design, integrity constraints, triggers, stored procedures, and advanced SQL queries.

## Project Objectives
- Design a normalized relational schema
- Implement business logic using triggers and stored procedures
- Demonstrate complex SQL queries and functions

## Files
- `sql/Online_Learning_Platform.sql` – Complete database implementation

## Advanced SQL Concepts Used
- Stored procedures for enrollment and grading workflows
- Triggers for enforcing business rules and data integrity
- User-defined functions for performance metrics
- Complex joins and nested subqueries
- Transaction control and constraint enforcement

## How to Run
```sql
CREATE DATABASE OnlineLearningPlatform;
USE OnlineLearningPlatform;
SOURCE sql/Online_Learning_Platform.sql;
